README !!!

1) in order to use the python script, you need:
- Python3
- urllib3
- bs4 (BeautifulSoup) 

Optional) to install Python3, go to 
https://www.python.org/downloads/
and download any Python 3 version, then follow the instructions from the installation package

2) in your MacOSX / Linux terminal, enter:
./run_install.sh
to install all dependencies

3) in your MacOSX / Linux terminal, enter:
python3 scrape_opensecret.py <"First Name"> <"Last Name">
to search for the donor data

4) No Windows support yet! Sorry :(

5) Have fun !!
